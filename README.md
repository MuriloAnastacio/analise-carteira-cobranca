# Análise de Carteira de Cobrança

Este projeto simula uma análise de carteira de cobrança com foco em inadimplência e recuperação.

## Objetivo
Analisar clientes em atraso e identificar:
- Percentual de inadimplência
- Aging list
- Valores em aberto
- Prioridade de cobrança

## Ferramentas
- Excel
- Tabela dinâmica
- Indicadores de cobrança

## Indicadores analisados
- Aging List
- Valor em atraso
- Percentual de inadimplência
- Clientes críticos

## Exemplo de Análise
A análise permite identificar clientes com maior risco e priorizar ações de cobrança.
